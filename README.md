# Game2 — 星糖消消乐

一个轻量级 Match-3 消消乐网页小游戏，用原生 HTML/CSS/JavaScript 实现。

## Features

- 8x8 棋盘
- 相邻交换
- 三连及以上自动消除
- 掉落补充与连锁计分
- 30 步限制
- 本地最高分记录
- 纯前端，可直接部署到 GitHub Pages

## Local Preview

```bash
python3 -m http.server 8765
# open http://127.0.0.1:8765
```

## Test

```bash
node test.js
```

## GitHub Pages

部署到 GitHub Pages 后，入口文件是 `index.html`。
